# niall-m.github.io

This repository contains my personal portfolio site, built as a static React application for deployment on GitHub Pages.

## Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

## Available Scripts

```bash
npm run build
npm run preview
npm run typecheck
```

## Deployment

The site is designed for static hosting on GitHub Pages and uses `HashRouter` so client-side routes continue to work without server-side route handling.

## Build Process

This site was built with an AI-assisted workflow using v0 for visual exploration and Codex for implementation support. Final content, architecture, code review, and deployment decisions were my own.
